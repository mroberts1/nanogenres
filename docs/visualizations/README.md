# Visualization usage guides
Short, opinionated readers' guides for each visualization in the aesthetics wiki — what's on screen, how to interact with it, and the patterns worth looking for.
- [Overlap graph](./overlap-graph.md) — `/graph`. How aesthetics relate to each other through shared canonical films.
- [Film cross-membership chips](./cross-membership-chips.md) — on every `/aesthetics/[slug]` page. How porous each aesthetic's canon is.
- [Catalog decade heatmap](./decade-heatmap.md) — `/decades`. Where in time each aesthetic's canon lives.
- [Genre → country Sankey](./genre-country-sankey.md) — bottom of every `/aesthetics/[slug]` page. How each aesthetic decomposes into traditional genres and countries.
- [Director constellation](./director-constellation.md) — `/directors`. Bipartite graph of directors who bridge multiple aesthetics.
- [Rating × canonicality scatter](./rating-canonicality-scatter.md) — `/scatter`. Every (film, aesthetic) pair plotted by appearance count and Letterboxd rating.
Each guide is short on purpose — read the relevant one before spending time interpreting the visualization, then come back to it if you spot something surprising.
